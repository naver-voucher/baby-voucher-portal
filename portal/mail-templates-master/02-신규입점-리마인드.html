[계약 솔루션 추가] #완료 여부 폼 기입
  • 계약 방법 : 커머스솔루션 마켓 접속 > 신청 가능한 바우처 솔루션 노출(판매자 당 1개 노출)
  • 계약 완료 기한 : 4월 27일 월요일 오후 5시까지
  • 계약 프로세스 안내
1. 커머스솔루션 마켓 접속: 스스판매자 센터 또는 커머스솔루션 직접 방문 > 000님께만 보이는 솔루션 내 '바우처 솔루션' 클릭 (베이비바우처 육아생필품)
ㄴ만약 솔루션이 보이지 않는 경우, 화면 우측 화살표를 눌러 확인해 주세요.

 


2. 솔루션 상품 안내 페이지 확인 : 상품상세 내 '추가하기' 클릭 시 다음 단계로 이동

  ※ 페이지 내 월 이용료는 0원으로 표시되며, 실제 정산 조건은 '바우처 부속합의서’에 별도 안내


3. 이용약관 동의 후 솔루션 추가: 솔루션 참여 필수 유의사항 & 이용약관/부속합의서 동의 후 '추가하기' 클릭



4. 신청 완료 안내 팝업: 별도 승인 절차 없이 바로 적용



5. 신청 완료 여부 확인: 스마트스토어 센터 > 커머스솔루션> '사용중' 탭에서 '베이비바우처 육아생필품' 확인


[판매자-회사 혜택 적용 동의 설정] #완료 여부 폼 기입

  • 동의 설정 방법 : 판매자 센터 > 고객혜택관리 > 혜택조회/수정 메뉴 상단 우측 > 판매자-회사(관리자) 혜택 적용 동의
  • 동의 완료 기한 : 4월 27일 월요일 오후 5시까지

※아래 3가지 체크포인트 모두 수락 필요
 ✅ 회사(관리자) 전액 부담 쿠폰 적용 동의
 ✅ 판매자-회사(관리자) 비용 분담 쿠폰 적용 동의
 ✅ 판매자-회사(관리자) 비용 분담 적립(포인트) 적용 동의
