# COS 메일 발송 툴 — 사용법 (Mac)

---

## 처음 사용하는 경우 — 최초 1회 설정

### STEP 1 — Python 설치 확인

1. 화면 오른쪽 상단 돋보기 아이콘 클릭 (또는 `Cmd + Space`)
2. `Terminal` 입력 후 엔터
3. 아래 명령어 입력 후 엔터

   ```
   python3 --version
   ```

4. `Python 3.x.x` 처럼 버전이 나오면 설치된 것 → STEP 2로 이동
5. 오류가 나오면 https://www.python.org/downloads/ 에서
   **Download Python** 클릭 → 설치 파일 실행 → 완료 후 STEP 2

---

### STEP 2 — 필수 라이브러리 설치

Terminal에서 아래 명령어 입력 후 엔터 (인터넷 연결 필요)

```
pip3 install requests openpyxl
```

`Successfully installed` 메시지가 나오면 완료.

---

### STEP 3 — 실행 권한 부여

Mac은 새로 받은 스크립트 파일에 실행 권한을 직접 부여해야 합니다.

1. Terminal 창에 아래를 입력하되 **엔터는 아직 누르지 마세요**

   ```
   chmod +x 
   ```

   (끝에 공백 한 칸 있음)

2. Finder에서 `실행.command` 파일을 Terminal 창 위로 **드래그 앤 드롭**
   - 파일 경로가 자동으로 입력됩니다

   ```
   chmod +x /Users/사용자명/Desktop/COS/실행.command
   ```

3. **엔터** → 아무 메시지도 안 나오는 게 정상

---

### STEP 4 — 실행

Finder에서 `실행.command`를 **더블클릭**

- Terminal 창이 잠깐 뜨면서 서버가 시작됩니다
- 2~3초 후 브라우저에서 자동으로 http://localhost:5000 열림

> **보안 경고 팝업이 뜨는 경우**
> `실행.command` 파일을 **우클릭 → 열기 → 열기** 클릭

---

### 다음부터는

`실행.command` **더블클릭**만 하면 됩니다. STEP 1~3은 다시 하지 않아도 됩니다.

## 4단계 발송 흐름

### 1단계 — 수신자 설정

**발신자 선택**

| 선택지 | 발신 주소 |
|---|---|
| 펫바우처 | dl_naverpet_2025@navercorp.com |
| 베이비바우처 | dl_babyvoucher@navercorp.com |

**테스트 이메일**: 3단계에서 먼저 받아볼 주소 (반드시 `@navercorp.com`)

**수신자 목록 — 2가지 모드**

| 모드 | 설명 |
|---|---|
| 일반 발송 | TO / CC / BCC 텍스트 입력 + 첨부파일 1개 |
| 수신자별 첨부 > 직접 입력 | 이메일 + 파일 버튼을 행마다 직접 설정 |
| 수신자별 첨부 > 엑셀 업로드 | 양식 다운로드 → 경로 입력 → 업로드 |

**수신자별 첨부 — 직접 입력 방법**

1. `수신자별 첨부파일` 탭 선택 → `직접 입력` 탭 확인
2. 이메일 입력 + `📎 파일 선택` 버튼으로 파일 첨부
3. `＋ 이메일 추가하기`로 행 추가

**수신자별 첨부 — 엑셀 업로드 방법**

1. `엑셀 업로드` 탭 선택
2. `📥 양식 다운로드` 클릭 → 엑셀 파일 열기
3. A열: 수신자 이메일, B열: 첨부파일 전체 경로 입력

   ```
   수신자이메일              첨부파일경로
   partner@example.com      /Users/사용자명/Downloads/A.zip
   brand@example.com        /Users/사용자명/Downloads/B.zip
   ```

   > 파일 경로 확인 방법: Finder에서 파일 선택 → `Cmd + Option + C` → 붙여넣기

4. 저장 후 `📤 목록 업로드` 버튼으로 업로드

---

### 2단계 — 메일 내용

- **메일 제목** 입력
- **본문 편집기**: 에디터(WYSIWYG) / HTML 직접 입력 / 텍스트 3가지 탭 제공

**예약 발송 설정**

하단 `예약 발송` 토글을 켜면 날짜/시간 선택기가 나타납니다.

- 현재 시간 기준 **최소 30분 이후** 시간만 설정 가능
- 체크 해제 시 즉시 발송

---

### 3단계 — 테스트 발송

1단계에서 입력한 **테스트 이메일**로 먼저 발송합니다.

- 수신자별 첨부 모드: 첫 번째 항목의 파일로 테스트
- 테스트 성공 후 `다음 → 최종 발송` 버튼 활성화

---

### 4단계 — 최종 발송

수신자 전체 목록 확인 후 `🚀 최종 발송` 클릭

- 수신자별 첨부 모드: 각 수신자에게 개별적으로 순차 발송
- 발송 진행률 및 결과 실시간 표시

---

## 발송 이력

- 발송 내역은 `app/history.json`에 자동 저장 (최대 100건)
- 좌측 사이드바에서 이전 발송 내역 클릭 시 해당 내용 복원
- 항목에 마우스 올리면 ✕ 버튼으로 삭제 가능
- 브라우저 캐시 삭제와 무관하게 영구 보관

## 자주 묻는 문제

| 증상 | 원인 / 해결 |
|---|---|
| 페이지가 안 열림 | `실행.command` 재실행 또는 `python3 --version` 터미널에서 확인 |
| 보안 경고 팝업 | `실행.command` 우클릭 → 열기 → 열기 |
| 더블클릭해도 반응 없음 | Terminal에서 `chmod +x` 명령어 다시 실행 (STEP 3) |
| 발송 실패 `COS 서버 연결 실패` | 사내 네트워크 또는 VPN 연결 확인 |
| `550 No such user` 오류 | 수신자 이메일 주소가 실제로 존재하지 않음 |
| 첨부파일 없음 오류 (엑셀 모드) | B열 경로가 실제 파일 경로인지 확인 |
| 예약 발송 오류 | 현재 시간 기준 30분 이후인지 확인 |
| 양식 다운로드 후 xlsx 안 열림 | Terminal에서 `pip3 install openpyxl` 후 재시작 |
